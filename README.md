📊 Retail Sales & Profit Analysis Dashboard (Power BI Project)

📌 Project Overview

This project presents an end-to-end business intelligence solution built using Power BI to analyze retail sales data. The dashboard provides actionable insights into sales trends, product performance, regional analysis, and customer segmentation.

The objective is to support data-driven decision-making by transforming raw data into meaningful visual insights.

🎯 Problem Statement

Businesses often struggle to track sales performance across different regions, product categories, and customer segments. This project aims to:

> Identify top-performing products and categories

> Analyze sales trends over time

> Detect profitability patterns

> Understand customer segmentation behavior

📂 Dataset Description

The dataset contains retail transaction data with the following key fields:

> Order Date & Ship Date

> Sales, Profit, Quantity

> Region, State, City

> Category & Sub-category

> Customer Segment

🧹 Data Cleaning & Transformation

Performed using Power Query Editor:

Removed duplicate and null values

Converted data types (Date, Numeric)

Extracted:

> Year

> Month Name

> Month Number

> Created custom column:

> Profit Status (Profit / Loss)

> Ensured proper sorting (Month Name by Month Number
